## 📲 手機「桌面捷徑小工具」安裝說明

透過 **PWA 技術**，您可以將它安裝在 iPhone 或 Android 手機的桌面上，擺脫瀏覽器的網址列，實現全螢幕原生 App 的沉浸體驗。

````carousel
### 🍏 iOS / iPhone 用戶安裝步驟
1. 使用 **Safari** 瀏覽器開啟 [bd_career_matrix_mobile.html](./bd_career_matrix_mobile.html)。
2. 點擊瀏覽器底部的 **「分享」按鈕**（向上箭頭的圖示）。
3. 在選單中向下滑動，選擇 **「加入主畫面」 (Add to Home Screen)**。
4. 點擊右上角的「新增」即可完成！桌面上將會出現 **「BD 職涯小工具」**。
<!-- slide -->
### 🤖 Android / 安卓用戶安裝步驟
1. 使用 **Chrome** 瀏覽器開啟 [bd_career_matrix_mobile.html](./bd_career_matrix_mobile.html)。
2. 點擊瀏覽器右上角的 **「三點選單」 ︙ **。
3. 選擇 **「新增至主畫面」** 或 **「安裝應用程式 (Install App)」**。
4. 依據系統提示點擊確定，應用程式即刻會以原生小工具形式常駐在您的桌面上！
````

## 💎 手機版專屬優化細節

為了達到 iOS/Android 原生應用等級的視覺與手感，我們進行了以下重構：

| 優化維度 | 原網頁版 | 手機 App 專屬版 | 優化帶來的價值 |
| :--- | :--- | :--- | :--- |
| **導航機制 (Navigation)** | 頂部按鈕與滾動時間軸 | **底部固定原生 Tab Bar (底欄)** | 符合拇指操作熱區，一鍵無縫切換探索、對照與自評。 |
| **階段選擇器** | 圓形數字進度條時間軸 | **分段控制器 (Segmented Control)** | 大面積可點擊區塊，單手滑動即可快速在初/中/高階間切換。 |
| **全景對照表** | 寬屏網格 Table 表格 | **智能折疊面板 (Accordion Cards)** | 解決手機橫向滾動表格的痛苦，按維度垂直展開對齊，直觀易讀。 |
| **自評 checklist** | 緊湊的勾選框 | ** thumb-friendly 觸控大卡片** | 將勾選框擴大為整列卡片點擊，加入微動態點擊與震動回饋感。 |
| **首頁安裝引導** | 無 | **智慧系統感應 Banner** | 自動識別 iOS/Android 系統並顯示專屬的中文引導說明，且可手動關閉。 |

## 📂 PWA 捷徑小工具檔案結構
我們已經在工作坊目錄中為您配備了最齊全的 PWA 架構檔案：
